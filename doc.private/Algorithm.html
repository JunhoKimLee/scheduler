Module Algorithm

module Algorithm: sig .. end

Ranking Algorithm that rules out invalid schedules and ranks schedules based on user preferences.

type event = Schedule.event

The type of event, representing one obligation with a set start and end time.

type t = Schedule.t

The abstract type of values representing a schedule.

type comparable_event = {
   event_object : event;
   start_time_min : int;
   end_time_min : int;
}

The type of comparable event, storing event, start time, and end time.

type t_valid = {
   events : t;
   start_time_valid : bool;
}

The type of valid schedule.

val get_start_time : comparable_event -> int

get_start_time cmp_event is the start time of cmp_event in minutes.

val time_to_min : Classes.time -> int

time_to_min time converts time to minutes in int from Classes record type.

val to_comparable_event : event -> comparable_event

to_comparable_event event converts event of type Schedule.event to type comparable_event. The output of this function is used to compare the event's time and duration with other events to determine if there is a time conflict.

val comparable_list : comparable_event list ->
       event list -> comparable_event list

comparable_list acc t is a comparable_event list from the schedule t.

val compare : comparable_event -> comparable_event -> int

compare a b is -1 when start time of event a is less than that of event b, 0 when equal, 1 when greater.

val sort_start_time : comparable_event list -> comparable_event list

sort_start_time cmp_list is a sorted comparable_event list from the least to the greatest start time values.

val start_time_lst : int list -> comparable_event list -> int list

start_time_lst acc cmp_list is an int list of start time values of event in cmp_list.

val is_duplicate : comparable_event list -> bool

is_duplicate cmp_list is true if cmp_list contains more than one event with the same start time. Otherwise, false.

val end_time_compare : comparable_event list -> bool

end_time_compare cmp_list is false if cmp_list contains at least one event that overlaps with another event. Otherwise, true.

val check_day_schedule : event list -> bool

check_day_schedule event_list is true if event_list satisfies both is_duplicate and end_time_compare, which means that the event list does not have any overlapping class times.

val check_valid_schedule : Schedule.t -> bool

check_valid_schedule t is true if the schedule t does not have overlapping class times on each day of the week. Otherwise, false.

val filter_valid_schedule : Schedule.t list -> Schedule.t list -> Schedule.t list

filter_valid_schedule acc t_list is the list of schedules acc filtered list of list of schedules t_list with the schedules that do not have overlapping class time.
