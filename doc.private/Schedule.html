Module Schedule

module Schedule: sig .. end

Representation of a schedule.

This module represents the configuration of classes stored in a schedule.

type event = {
   course_name : string;
   description : string;
   course_id : Classes.course_id;
   credits : int;
   subject : string;
   catalog_number : int;
   event_type : string;
   section_id : Classes.section_id;
   section_number : string;
   meeting_id : Classes.meeting_id;
   start_time : Classes.time;
   end_time : Classes.time;
   instructors : string list;
   facility : string;
   building : string;
   days : Classes.day list;
}

The type of event, representing one obligation with a set start and end time.

type permutation = {
   course_id : Classes.course_id;

type t = event list

The abstract type of values representing a schedule.

val make_event : Classes.meeting_id ->
       Classes.section_id -> Classes.course_id -> Classes.t -> event

make_event m s c r makes an event data type of meeting m in section s in class c in roster r.

val add_section : Classes.section_id ->
       Classes.course_id -> Classes.t -> event list -> event list

add_section s c r t is the schedule t with the addition of section s of class c in roster r. Raises: ClassNotFound c if c is not a valid course id. SectionNotFound s if s is not a valid course id.

val empty : 'a list

empty is the representation of the empty schedule.

val is_empty : 'a list -> bool

is_empty t is true if t is empty. Else, false.

val size : 'a list -> int

size t is the number of sections in schedule t.

val peak : 'a list -> 'a

peak t gives an element of t.

val get_events : 'a -> 'a

get_events t is the list of events that represents all events in schedule t.

val get_day : Classes.day ->
       event list -> event list -> event list

get_day d a c is a list of events from c that occurs on day d, appended to a.

val get_monday : event list -> event list

get_monday t is the list of events that represents all events in schedule t on Monday.

val get_tuesday : event list -> event list

get_tuesday t is the list of events that represents all events in schedule t on Tuesday.

val get_wednesday : event list -> event list

get_wednesday t is the list of events that represents all events in schedule t on Wednesday.

val get_thursday : event list -> event list

get_thursday t is the list of events that represents all events in schedule t on Thursday.

val get_friday : event list -> event list

get_friday t is the list of events that represents all events in schedule t on Friday.

val perm_to_sched : (Classes.course_id * Classes.section_id list) list ->
       permutation list ->
       (Classes.course_id * Classes.section_id list) list list

perm_to_sched l p is a list of schedule configurations where each config is a list of sections for each class, section permutations in a tuple of course id and type permutation from p. Appended to l. Example: [(593554,[10601;19118]);(593554,[10601;19118])];[(593554,[10601;19118]);(593554,[10601;19118])]

val parse_sections : Classes.section_id list ->
       string ->
       Classes.section_id list ->
       Classes.course_id -> Classes.t -> Classes.section_id list

parse_sections l req s c r is the list of section id's from s from course c in roster r that satisfy being of section type req. Appended to list l.

val list_of_list : Classes.section_id list list ->
       string list ->
       Classes.section_id list ->
       Classes.course_id -> Classes.t -> Classes.section_id list list

list_of_list a req s c r is a list of lists, where each list of list is a list of section id's that require one part of req, where the sections are drawn from s_list from course c in roster r. Results appended to a. Example: [10601;10602];[10603,10604]

val generate_perms : 'a list -> 'a list list -> 'a list list

generate_perms a p is a list of list of section id's, representing all possible permuations of sections from list of list of section id's, where each list within a list is all the section id's for one required component. Appended to a.

val class_permutations : Classes.course_id -> Classes.t -> permutation

class_permutations c r is a record of type permutation containing all section permutations of class c in roster r.

val list_of_perm : permutation list ->
       Classes.t -> Classes.course_id list -> permutation list

list_of_perm l r c is a list of all class permutations of type permutation, from list of class id's c of roster r, appended to list l.

val schedule_maker : Classes.t -> event list list

schedule_maker r is a list of schedules of type t, that represents all possible schedules as a result of permutations of classes in c.
