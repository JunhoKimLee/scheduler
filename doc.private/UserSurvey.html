Answer collection for user survey questions.

The abstract type of values representing user survey answers.

   final_semester : string;

}

The output type of UserSurvey.

val init_state : t

init_state is the initial state of survey response where the answers to all of the questions are set as empty

val get_semester : t -> string

get_semester st is a string of semester for current state st.

val get_classes : t -> (string * string) list

get_classes st is an association list of classes for current state st.

val final_output : t -> t_output

final_output st is a record with semester of tuple list of classes inputted in final state st

val check_length : string list -> bool

check_length tl is true if for the string list tl with two elements, the length of first string (ex."CS") is between 2 and 5 and the length of second string (ex."3110") is 4. False otherwise.

val check_type : string list -> bool

check_type tl is true if for the string list tl with two elements, the first string has only non-numbers, and the second string has only numbers. False otherwise.

val is_valid_class : string list -> bool

is_valid_class tl is true if the inputted class is a valid format of class name. It is false otherwise.

val is_valid_class_st : t -> bool

is_valid_class_st st is true if the inputted class is more than 3 classes and less than 9 classes, which generally satisfies the minimum and maximum credits required or available.

val check_sem : string -> bool

check_sem sem is true if the inputted string sem is either "SP" or "FA". Otherwise, it is false.

val check_num : string -> bool

check_num hd is true if the inputted number hd is a 1-2 digit number. Otherwise, it is false.

val is_valid_sem : string list -> bool

is_valid_sem tl is true if both the letter part and the number part of the input tl has the correct format. Otherwise, it is false. This will be checked through check_sem and check_num.

val is_valid_sem_st : t -> bool

is_valid_sem_st st is true if the semester of current state st is not an empty string.

val take_sem : t -> string list -> t

take_sem st tl is the user attempting to take a semester input tl as their answer to the survey question. If the user has not answered the question yet and has an empty string as one's answer, the user will take the content of tl as the input. Otherwise, the user will have the same previous answer as the input and ignore the incoming one.

val delete_sem : t -> string list -> t

delete_sem st tl is the user attempting to delete the previous input tl for the semester prompt. If the user is attempting to delete the pre-existing input, then the user will successfully empty out the reponse for this particular question. Otherwise, the input will remain in the state, ignoring the incoming delete command.

val take_class : t -> string list -> t

take_class st tl is the user attempting to take a class tl as their answer to the survey question. The string list of one class (ex. "CS";"3110") is concatenated to the classes_input of st.

val delete_class : t -> string list -> t

delete_class st tl is the user attempting to delete a class tl as their answer to the survey question. The classes_input of st is a new string list list that has all the elements except tl.

val print_class_helper : string list -> string list list -> string list

print_class_helper acc classes is the string acc that concatenates all the class_ids in the string list listclasses

val print_sem : t -> string

print_sem st is the string that reveals the current input in the state st for semester question prompt.

val print_class : t -> string

print_class converts classes_input of st into a string with the correct format to be printed in the prompt.

val prompt_routine : 'a -> 'b

prompt_routine st prompts user to answer routine question and updates corresponding information to st. It also handles any commands that are written.

val prompt_class : t -> t

prompt_class st prompts user to answer class question and updates corresponding information to st. It also handles any commands that are written.

val prompt_semester : t -> t

prompt_semester st prompts user to answer semester question and updates corresponding information to st. It also handles any commands that are written.
